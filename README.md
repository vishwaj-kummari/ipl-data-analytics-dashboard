# ipl-data-analytics-dashboard
# Overview
This project presents an interactive IPL Performance Dashboard built to analyze player and team performances in the Indian Premier League (IPL). The dashboard provides insights into batting, bowling, team win percentages, and team loss percentages through dynamic visualizations and filters.

# Key Metrics
Total Matches: 5,720
Total Runs: 7,263
Total Wickets: 2,245
Average Strike Rate: 140.88

# Features
Interactive player and team filters (Slicers)
Batting performance analysis
Bowling performance analysis
Team performance comparison
KPI Cards for quick insights
Dynamic charts and visualizations
